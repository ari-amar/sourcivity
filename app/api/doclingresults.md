## Description

The 100G QSFP28 transceiver modules are designed for use in 100 Gigabit Ethernet links over multimode fiber. They are compliant with the IEEE 802.3bm 100GBASE-SR4 and CAUI-4. Digital diagnostics functions are available via the I2C interface as specified by the QSFP28 MSA.
